## Abstract
본 Plugin은 RGB Line Project의 핵심 기능을 담고 있다

Plugin은 ```RGBLineCoreLib.dll```과 ```RGBLineCoreLib_ForEditor.dll```의 두 가지 형태로 제공된다

## Usage
> [!IMPORTANT]\
> Editor의 경우 필히 ```RGBLineCoreLib_ForEditor.dll```를 사용할 것
